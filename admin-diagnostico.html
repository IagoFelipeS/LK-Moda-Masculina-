📋 Checklist de Diagnóstico

  1. ✅ Firebase está carregando? (verificar abaixo)
  2. ✅ Produtos estão sendo salvos no localStorage?
  3. ✅ O site consegue sincronizar?
  4. ✅ Produtos aparecem em index.html?
  5. ✅ Produtos aparecem em produtos.html?

🚀 Para Resolver o Problema

  1. Se Firebase está offline: Produtos serão salvos localmente e sincronizarão quando online
  2. Se produtos não aparecem no site:
    • Clique em "Sincronizar para Site" acima
    • Recarregue a página (F5)
    • Limpe o cache do navegador (Ctrl+Shift+Delete)
  3. Se ainda não funciona:
    • Abra console (F12 → Console)
    • Procure por mensagens de erro vermelho
    • Tire um print screen e compartilhe com suporte
